# Mobile App
A modern, intuitive mobile application for managing your vehicle maintenance, service history, and driving analytics.

## Features

- 🚗 Vehicle Management
    - Track multiple vehicles
    - Store vehicle documentation
    - Maintenance scheduling
- 📊 Analytics Dashboard
    - Fuel efficiency tracking
    - Driving behavior insights
    - Cost analysis
- 🔧 Service Records
    - Digital service history
    - Maintenance reminders
    - Service provider directory

## Getting Started

### Prerequisites

- iOS 14.0 or later
- Android 8.0 or later
- Active internet connection

### Installation

```bash
# Clone the repository
git clone https://github.com/notionmotors/mobile-app.git

# Install dependencies
cd mobile-app
npm install

# Run the development server
npm start

```

## Tech Stack

- React Native
- TypeScript
- Redux
